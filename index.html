About the game
Built for speed.
Engineered for FAANG.

APEX RACER X is a browser-native F1 hill-climb racing game built entirely from scratch — no Unity, no Godot, no game engine. Every system was hand-crafted: a custom delta-time physics engine, a Catmull-Rom spline terrain generator, a synthesised Web Audio engine with 7 distinct sounds, and a live weather system with 6 states.

Race against 3 AI rivals powered by a Finite State Machine with 5 states — WARMUP, CHASE, ATTACK, DEFEND, RECOVER. Each rival has a unique personality: K. Volkov is an aggressive attacker, E. Müller is a technically precise defender, L. Nakamura is a balanced threat.

The AI Crew Chief streams live tactical advice via the Claude API, reading your real telemetry — tyre temperatures, grip levels, lap time, weather state — and speaks it aloud through Web Speech synthesis. Challenge friends with WebRTC ghost racing, or race live P2P with automatic room codes.

Core systems
9 engines. Zero game frameworks.
⚙️
Custom Physics
Delta-time normalised engine. Catmull-Rom spline terrain. Slope gravity, grip stack, landing detection. Frame-rate independent at 60Hz, 120Hz, 144Hz.
🤖
LLM Crew Chief
Streaming Claude API reads your live telemetry — tyres, speed, weather, lap time — and gives real tactical radio advice in your ear via Web Speech synthesis.
🧠
FSM Rival AI
3 rivals with personalities. 5 FSM states: WARMUP → CHASE → ATTACK → DEFEND → RECOVER. Rubber-banding keeps races tight. Skill scales per level.
📡
WebRTC Multiplayer
Pure WebRTC P2P — no PeerJS, no server. Automatic room codes (APEX-WOLF-427). Ghost racing: record your best run, share via URL, race your own ghost.
🎵
Web Audio Engine
7 fully synthesised sounds from OscillatorNode + BiquadFilterNode. Engine pitch tied to RPM, turbo whine, tyre screech, rain ambience. Zero audio files.
🌩
Live Weather
6 weather states from SUNNY to STORM. Grip drops from 100% to 43%. Rain particles, lightning, fog, wind. AI crew chief reacts to weather changes instantly.
